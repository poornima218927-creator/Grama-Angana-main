# 🌾 Grama-Angana – Rural Service Android Application

Grama-Angana is an Android application designed to support rural communities by providing easy access to government schemes, local services, announcements, and community information through a simple and user-friendly mobile platform.

---

## 📱 Project Overview

The main objective of this project is to digitally connect rural people with important services and information. The application helps users:

* Access government schemes and welfare programs
* View village and community updates
* Receive important notifications and alerts
* Explore local services in one place
* Improve digital awareness in rural areas

---

## ✨ Features

* 🔐 User Login and Registration
* 🏛️ Government Scheme Information
* 📰 Local News and Updates

## ⚙️ Installation & Setup

### Step 1: Download the Project

* Clone the repository:

```bash
git clone https://github.com/poornima218927-creator/Grama-Angana.git
```

OR

* Download the ZIP file and extract it.

---

### Step 2: Open in Android Studio

1. Open Android Studio
2. Click **Open Project**
3. Select the extracted project folder
4. Wait for Gradle Sync to complete

---

### Step 3: Run the Application

1. Connect an Android device or start an emulator
2. Click the ▶ Run button
3. The application will build and launch automatically

## 🎯 Problem Statement

People in rural areas often face difficulties accessing government services, local information, and digital resources. Grama-Angana provides a centralized mobile platform that improves communication, accessibility, and awareness for rural communities.
